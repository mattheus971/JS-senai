Atalhos do VS Code

Atalhos gerais

Vamos ver alguns atalhos legais do VS Code que vão deixar nossa vida mais facil.

CTRL=ESPAÇO: Chama o autocompletar de um texto.

ALT+Z: Quebra automatica de linha.

ALT+SETAS cima/baixo: Move a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor está.

SHIFT+ALT+SETAS cima/baixo: Copia a seleção para cima/baixo, se nada estiver selecionado, copia a linha em que o cursor esta.

CNTRL+; : Comentário/descomentário
